@brief: In this module, various simple independent (separated) and useful functions and structures are introduced. It is intended to use them in other classes of this project as well as in stand alone programs.

@brief: This is a flag, which switch on debug printing. Comment this line if you do not want to print the debug communicates.
@sign: define ROJ_DEBUG_ON

@brief: This is a flag, which switch on a progress. Comment this line if you do not want to print the progress.
@sign: define ROJ_PROGRESS_ON

@brief: This is a const string, which defines the version number. Please do not change it manually. In order to change modify VERSION file and run 'make version'
@sign: define ROJ_VERSION "0-41"

@brief: This is the alias to 2 pi value.
@sign: define TWO_PI 6.2831853071795864769252866

@brief: This is Euler number.
@sign: define E_NUMBER 2.7182818284590452353602975

@brief: Blackman-Harris window code.
@sign: define ROJ_BH_WINDOW 0

@brief: ODE filter type code
@sign: define ROJ_ODE_FILTER 1

@brief: please comment it if your consol does not support color text
@sign: define ROJ_COLOR_TEXT

@brief: debug prompt
@sign: define ROJ_DEBUG_PROMPT "\033[1;31m(debug)\033[0m"

@brief: This is a structure for roj_complex_signal configuration (as well as roj_fourier_spectrum and others).
@sign: struct roj_signal_config
unsigned int length
double start
double rate

@brief: This is a structure for roj_real_array configuration.
@sign: struct roj_array_config
unsigned int length
double min
double max

@brief: This is a structure for roj_real_matrix configuration.
@sign: struct roj_image_config
roj_array_config frequency
roj_array_config time

@brief: The routine prints info about the project.
@sign: void print_roj_info ()

@param [in] a_major: A major part of the version.
@param [in] a_minor: A minor part of the version
@sign: void require_roj_version (int a_major, int a_minor)

@brief: This function shoud be called if an error is detected.
@param [in] a_info: A message, statement for printing in stderr.

@brief: This function shoud be called if a warning is detected.
@param [in] a_info: A message, statement for printing in stderr.
@sign: void call_warning (const char* a_info)

@brief: This function shoud be called if any info is detected.
@param [in] a_info: A message, statement for printing in stderr.

@brief: This function shoud be called for marker printing.

@sign: void print_marker (const char* a_info)

@param [in] a_info: A printing label.
@sign: void print_progress (int a_curr, int a_len, const char *a_info)

@brief: This function verify the correctness of an array config. It call error if not.

@brief: This function verify the correctness of an image config. It call error if not.

@sign: void verify_config(roj_image_config a_conf)

@brief: This function returns random number with normal probability using Box-Muller transform (G. E. P. Box and Mervin E. Muller, A Note on the Generation of Random Normal Deviates, The Annals of Mathematical Statistics (1958), Vol. 29, No. 2).
@param [in] a_sigma: Standard deviation.
@return: Random number.
@sign: double rand_gauss (double a_sigma)

@brief: This function returns factorial of the argument.
@param [in] a_number: The number for which factorial is roj_calculated.
@return: Factorial value.
@sign: int calc_factorial (int a_number)

@brief: This function returns binominal coefficient of two arguments.
@param [in] a_kval: A first argument.
@param [in] a_nval: A second argument.
@return: Binominal coefficient.
@sign: double calc_binominal (int a_kval, int a_nval)

@brief: This function returns Eulerian number of two arguments.
@param [in] a_kval: A first argument.
@param [in] a_nval: A second argument.
@return: Eulerian number.
@sign: double calc_eulerian (int a_kval, int a_nval)

@brief: This function returns one coefficient of blackman-Harris window.
@param [in] a_nr: An index of returned coefficient.
@param [in] a_len: A window length (in coefficients).
@param [in] a_d_order: An order of derivative.
@return: The coefficient value.
@sign: double calc_blackman_harris (int a_nr, int a_len, int a_d_order)

@brief: This routine is used for sorting and compares two double values.
@param [in] a_arg1: A pointer to double value.
@param [in] a_arg2: A pointer to double value.
@return: The result of the comarison which is compatible with the 'qsort' function.
@sign: int value_comparer (const void * a_arg1, const void * a_arg2)

@brief: This function calculates the determinat of 4x4 matrix.
@param [in] a_matrix: a given matrix.
@return: calculated determinant.
@sign: double calc_4x4_det (double a_matrix[4][4])
